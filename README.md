
## 技术栈

react@16.* + redux@3.* + react-router@4.* + webpack@3.* + axios@0.18.0 + less@2.7.3 + antd@3.8.2



## 项目运行


```
git clone https://github.com/hanyueqiang/create-react-app.git  

cd react （进入当前的项目）

yarn  i (安装依赖包)

yarn start (运行本地开发环境)

yarn build (打包，部署)

```

服务端数据使用的easy-mock
